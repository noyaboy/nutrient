## 🔴 CRITICAL ISSUES (立即修正)

### 1️⃣ **碘缺乏症 — 假設性風險**
**狀況:**
```
當前策略: 碘鹽 1g (20-33mcg) + 海苔/紫菜 1-2 片 (12-43mcg/片)
典型實際攝取: 33 + 50 = 83mcg
RDA 目標: 150mcg
⚠️ 缺乏量: 67mcg (44% 缺陷)
```

**生理風險:**
- 甲狀腺功能減退 (低T4/T3)
- 認知能力下降 (與你的「新技能學習」計畫衝突)
- 甲狀腺腫大 (長期)
- 因日常十字花科蔬菜 (花椰菜每日) 含有促甲狀腺激素物質 (goitrogens)，缺碘風險**倍增**

**為什麼採購避開昆布?** ✅ 正確決定
- 昆布碘含量極度變異 (1-2g = 300-6000mcg) → 精密控制不可能
- 但這個決定**無意中創造了缺陷**

**❌ 必須修正:**
- **選項 A (建議):** 新增補充劑 — iHerb 購入 **75-100mcg 碘補充劑** (月成本 $5-10)
  - NOW Foods 或 Solgar 碘片 (150mcg，隔日取一半)
  - 或每日 75mcg 劑量產品
- **選項 B:** 海苔攝取增加至 **3-4 片/日** (而非 1-2 片)
  - 保留 4-11mcg 的 UL 安全邊界 (vs 1100mcg UL)
  - 但仍存在變異性風險
- **選項 C (不建議):** 恢復受控昆布 — 一種精確劑量的產品 (需找新來源)

**建議:** 選項 A (最科學、成本最低、精密度最高)

### 2️⃣ **鈣缺乏症 — 食物優先策略脆弱**
**狀況:**
```
當前策略: 希臘優格 200-300g + 豆腐 100g (2x/週) + 深綠蔬菜
典型攝取: 200-300mg (希臘優格) + 150mg (豆菜) + 50-100mg (蔬菜) = 400-530mg
RDA 目標: 1000mg (成年男性)
⚠️ 缺乏量: 470-600mg (50-60% 缺陷)
```

**長期風險:**
- 骨密度下降 (strength training 無法補償缺鈣)
- 50L冷箱容量限制 → 優格腐壞風險 → 依賴性不可靠

**為什麼現有策略失敗:**
- 計畫假設「完美優格遵守」(每日 300g) — 現實中會有遺漏日
- D3 + K2 已備妥支持骨礦化，但**無足夠鈣基質就失效**

**❌ 必須修正:**
- **選項 A (強烈建議):** 新增每日 **500mg 鈣補充劑** (Nature Made 已在採購)
  - 改為「每日補充」而非「backup only」
  - 午餐隨餐取用 (碳酸鈣需胃酸)
  - 與晚餐 Mg/Zn **不衝突** (7hr 間隔保護)
- **選項 B:** 強制每日 300g 優格紀律 + 高鈣替代品追蹤 (複雜)
- **結合兩者:** 優格 200g (固定) + 補充劑 300mg (彌補缺口)

**建議:** 選項 A — 將 Nature Made Ca 500mg 從「backup」改為「每日午餐固定」

### 6. **熱量目標 vs 巨量營養不一致**
**狀況:**
```

  - 蛋白 122-132g = 488-528 kcal
  - 脂肪 80-90g = 720-810 kcal
  - 碳水 (假設 5-6g/kg) = 365-438g = 1460-1752 kcal
  總計: 2668-3090 kcal
❌ 缺少 10-600 kcal (取決於碳水位置)
```

**問題:**
- 若目標是 3100-3400，碳水需要 6.5-7.5g/kg 才夠
- 計畫未明確說明這額外碳水來自哪裡

**✅ 建議:**
- 選擇 A: 下調目標為 **2800-3000 kcal** (與現有巨量對齊)
- 選擇 B: 上調碳水至 **6.5g/kg = 474g** (於 3100-3400 kcal 工作)
- 選擇 C: 在計畫中明確列出「額外碳水來源」(例如: 訓練日 +2-3 碗米飯)
- **建議:** 選擇 A (現有策略能量充足；3400 kcal 設定可能來自未來增重階段)
