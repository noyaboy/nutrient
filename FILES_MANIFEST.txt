USAGE SUMMARY

Quick Start (60 seconds):
```bash
pip install pdfplumber
python parse_procurement_pdf.py 健康長壽追蹤器採購.pdf
head -100 procurement_output.json | python -m json.tool
```

Common Commands:
```bash
# Parse with custom output
python parse_procurement_pdf.py 健康長壽追蹤器採購.pdf -o my_data.json

# List PDF pages
python parse_procurement_pdf.py --list-pages 健康長壽追蹤器採購.pdf

# Run 6 example scripts
python example_usage.py

# Validate JSON output
python -m json.tool procurement_output.json

# Search for specific product
python -c "
import json
with open('procurement_output.json') as f:
    data = json.load(f)
results = [p for p in data if 'Kirkland' in p.get('product_name', '')]
for p in results[:3]:
    print(p['product_name'], '-', p.get('price'))
"
```
